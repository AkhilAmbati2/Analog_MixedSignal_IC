# Master's Thesis

This folder contains my Master's thesis work completed at TU Chemnitz,
focused on advanced analog IC design methodology and implementation.

## Thesis Title
**Methodology to Optimize the Low-Dropout Regulator (LDO)**

## Key Contributions
- Python-based design automation and trade-off analysis
- Block-level optimization for power, stability, and robustness
- Cadence Virtuoso schematic, layout, and verification flow

## Tools
- Cadence Virtuoso (schematic & layout)
- Spectre (simulation)
- Python (NumPy, Matplotlib)

## Contents
- `report/` – Final thesis report (PDF)
- `figures/` – Key plots and diagrams
- `python/` – Design and sizing scripts

> Note: No proprietary PDKs or model files are included.
